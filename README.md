## About project

**Metro: Underground Echo** is 3D first person view game powered by unity engine.

## Game Description

The game about a man, that go down to metro and found lost catacombas. It seems like something weird leaves here, but you can't prove it. 

## IT HE#RS YOU

Current version has:

- 2 locations
- Different menus
- Stamina + health
- Monster with simple AI
